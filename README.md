# Pizza Sales Analysis Dashboard 🍕

## Project Overview
This project analyzes pizza sales data using **SQL** for querying and **Power BI** for visualization.  
The objective is to identify sales trends, customer behavior, and business insights that can help improve business performance.

---

## Tools Used

- **SQL (MySQL)**
- **Power BI**
- **Excel / CSV Dataset**
- **GitHub**

---

## Dataset Information

The dataset contains four tables:

- **Orders** → Stores order date and time  
- **Order Details** → Stores quantity of pizzas ordered  
- **Pizzas** → Pizza price and size details  
- **Pizza Types** → Pizza category and ingredients  

---

## SQL Analysis Performed

Business questions solved using SQL:

- Total Revenue Generated  
- Total Number of Orders  
- Average Order Value  
- Top 5 Best Selling Pizzas  
- Category Wise Sales Analysis  
- Sales by Pizza Size  
- Monthly Sales Trend  
- Peak Order Time Analysis  
- Top Revenue Generating Pizza Types  

---

## Power BI Dashboard Features

The dashboard includes:

- Revenue KPI  
- Total Orders KPI  
- Average Order Value KPI  
- Sales by Pizza Category  
- Sales by Pizza Size  
- Daily Sales Trend  
- Top Selling Pizza Types  
- Revenue Trend Analysis  

---

## Key Business Insights

### 1. Classic Category generated the highest revenue.

### 2. Large size pizzas were ordered most frequently.

### 3. Weekend sales were higher than weekday sales.

### 4. Evening hours showed peak customer demand.

### 5. A small number of pizza types contributed most of the total revenue.
